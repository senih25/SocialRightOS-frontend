# M2.0B — Controlled Beta Release Checklist

## Ürün ve UX

- [ ] Ana sayfa, start ve iki pilot route desktop/mobile smoke geçti.
- [ ] Loading tekrar gönderimi engelliyor.
- [ ] Validation, API/network, unexpected ve `UNAVAILABLE` birbirinden ayrılıyor.
- [ ] Sonuç resmî karar olmadığını, kaynağı ve sonraki adımı gösteriyor.
- [ ] Route/endpoint/payload/CTA/analytics/threshold regression sayıları sıfır.

## Erişilebilirlik

- [ ] Klavye focus görünür.
- [ ] Label/description/error association doğrulandı.
- [ ] Error ve result canlı bölgeleri çelişkili metin üretmiyor.
- [ ] Semantik başlık/listeler ve boş bölüm omission doğrulandı.
- [ ] Kritik erişilebilirlik bulgusu 0.

## Güven, privacy ve legal

- [ ] KVKK/gizlilik, kullanım koşulları, yasal sınır, kaynak-güncellik ve iletişim sayfaları mevcut.
- [ ] `HUMAN_LEGAL_REVIEW_REQUIRED` durumu kayıtlı.
- [ ] Özel nitelikli kişisel veri girmeme uyarısı görünür.
- [ ] Client bundle secret, token logging, PII analytics, raw carrier ve hassas localStorage yok.
- [ ] External link ve HTML kullanımı güvenli.

## SEO/GEO ve performans

- [ ] Public route'larda unique title/description/canonical doğrulandı.
- [ ] robots/sitemap/indexing kuralları ortama uygun.
- [ ] OG/Twitter ve doğrulanmış structured-data çıktısı güvenli.
- [ ] Heading hierarchy ve sitemap route bütünlüğü geçti.
- [ ] Build çıktısı ve büyük route chunk'ları kaydedildi.

## Operasyon

- [ ] CI typecheck/lint/test/build/diff/secret kapılarını çalıştırıyor.
- [ ] Release SHA ve environment marker mevcut.
- [ ] Preview smoke PASS veya erişim blocker'ı açıkça kayıtlı.
- [ ] Rollback runbook/provası kayıtlı.
- [ ] P0 açık 0.
- [ ] Production deploy/domain/DNS/billing değişikliği 0.

## Final karar

- [ ] `READY_FOR_CONTROLLED_BETA`
- [ ] `READY_WITH_NON_BLOCKING_FINDINGS`
- [ ] `NOT_READY`

Yalnız biri işaretlenir; production deploy ayrıca onaylanmadan yapılmaz.
