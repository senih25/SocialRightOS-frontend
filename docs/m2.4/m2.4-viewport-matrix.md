# M2.4 Viewport Matrix

Yerel standalone production build, gerçek Chromium layout engine ve 1px element-bound toleransı ile ölçüldü.

| Viewport | Route kontrolü | Maksimum overflow | Sonuç |
|---|---:|---:|---|
| 320×568 | 9 | 0px | PASS |
| 360×800 | 9 | 0px | PASS |
| 375×812 | 9 | 0px | PASS |
| 390×844 | 9 | 0px | PASS |
| 393×873 | 9 | 0px | PASS |
| 412×915 | 9 | 0px | PASS |
| 768×1024 | 9 | 0px | PASS |
| 1440×900 | 9 | 0px | PASS |

Route seti: ana sayfa, GSS, 65 yaş, Gizlilik/KVKK, Kullanım koşulları, Yasal uyarı, Kaynak ve güncellik, İletişim ve 404. Toplam 72 kontrolün tamamında `scrollWidth <= clientWidth`, viewport dışı element sayısı 0 ve beklenen HTTP durumu doğrulandı.

GSS ve 65 yaş sentetik sonuç yüzeyleri 390×844'te ayrıca üretildi. Sonuç kartı, CTA/kaynak bağlantısı ve erişilebilir status yüzeyi taşmadan görünür kaldı. 28 öğelik klavye sırasında focus ring görünür ve viewport içinde kaldı.
