# M2.3 Security and Privacy Report

## Runtime ve response güvenliği

- HTTPS ve expected CSP, `X-Content-Type-Options: nosniff`, Referrer-Policy ve Permissions-Policy mevcut.
- Vercel error-level ve HTTP 500 log sorguları sonuç döndürmedi.
- Browser console error/warning sayısı 0.
- Sekiz public JS adayı için `.map` isteği kontrol edildi; exposed source map 0.
- `/_next/static/` canonical redirect sonrası `/_next/static` HTTP 404; directory listing yok.
- Negatif response'larda stack, TypeError, source path, raw payload, carrier, secret veya PII yok.
- UI sonuçlarında raw backend response ve validation carrier görünür değil.

## Analytics ve privacy

- Runtime'da analytics provider script'i yok.
- `window.dataLayer` yok ve external analytics network request görülmedi.
- Browser ağ listesi yalnız same-origin Next.js/API isteklerinden oluştu.
- Sentetik assessment request dışında veri gönderilmedi.
- Ad, soyad, TC, telefon, e-posta, sağlık verisi, serbest metin veya gerçek form verisi kullanılmadı.
- Source allowlist testleri 195/195 suite içinde analytics envelope'un yalnız izinli event/alanlardan oluştuğunu doğruladı.
