# M2.2 Negative Payload Test Matrix

Tüm negatif HTTP vakalarında beklenen ortak sözleşme: HTTP 400, `invalid_request`, 500 yok, TypeError/stack/internal path/raw payload/validation carrier/secret/PII yok.

| Sınıf | Örnek | Beklenen |
|---|---|---|
| No body | body yok | Güvenli 400 |
| Empty body | boş byte dizisi | Güvenli 400 |
| Invalid JSON | `{` | Güvenli 400 |
| JSON null | `null` | Güvenli 400 |
| JSON string | `"not-an-object"` | Güvenli 400 |
| JSON array | `[]` | Güvenli 400 |
| Empty object | `{}` | Güvenli 400 |
| Missing eligibility root | `{ "eligibility": null }` | Canonical `benefit_code`/`facts` yok; güvenli 400 |
| Required property null | `benefit_code: null` | Güvenli 400 |
| Required nested missing | `facts` yok | Güvenli 400 |
| Required nested null | `facts: null` | Güvenli 400 |
| Unexpected nested array | `facts: []` | Güvenli 400 |
| Wrong nested type | fact value array/object | Güvenli 400 |
| Unsupported benefit | `TR_UNKNOWN` | Güvenli 400; sessiz GSS default yok |
| Unknown root field | geçerli payload + ek root scalar | Ek alan normalize çıktısından atılır; geçerli akış korunur |
| Oversized structurally valid | 1.000 sentetik scalar fact uzantısı | Crash yok; geçerli GSS akışı |

Pozitif regresyonlar:

| Akış | Beklenen |
|---|---|
| TR_GSS canonical payload | HTTP 200, `benefit_id=TR_GSS` |
| TR_OLD_AGE_PENSION canonical payload | HTTP 200, `benefit_id=TR_OLD_AGE_PENSION` |
| Mevcut API error ayrımı | `ApiClientError` sözleşmesi değişmez |
